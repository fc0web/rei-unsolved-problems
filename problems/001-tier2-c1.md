## Statement (informal)

Collatz 軌道の peak 値が **{9232, 13120, 4372, 1672}** (top 4 attractor) のいずれでもない n (ISOLATED と呼ぶ) に対して、軌道長 K(n) が常に `K(n)·100 ≤ 444·bitLen(n)²` を満たすか?

## Statement (formal)

```
C1 (ISOLATED universal):
  ∀ n > 235, odd,  peak(n) ∉ {9232, 13120, 4372, 1672}
    ⇒  K(n)·100 ≤ 444·bitLen(n)²
```

Lean 4:
```lean
axiom C1_isolated_universal :
  ∀ n > 235, Odd n → peak n ∉ ({9232, 13120, 4372, 1672} : Set ℕ) →

```

## Background

Rei-AIOS Paper 106 "Conditional Complete Proof" で tier2 bound `K(n)·100 ≤ 444·bitLen(n)²` を提示. このうち **n ≤ 10⁸ は empirical に 0 violations** (8.46M integers) だが、**∀ n** は 3 Collatz-equivalent axioms (C1/C2/C3) に conditional.

C1 は 4 primary attractor 以外の軌道を全て支配できるかを問う.

## Why it matters

- C1 成立 ⇒ ISOLATED 部分 (> 99% of n) が closed
- C2 (tail ∀ n > 10⁸) と C3 (mod-8 residual) と併せて tier2 が unconditional になる
- **tier2 unconditional ⇒ Paper 106 が structural complete proof** として認定される可能性
